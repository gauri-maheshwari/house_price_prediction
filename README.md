 
🏠 House Price Prediction Using Machine Learning



## Author

Gauri Maheshwari

## Project Overview

This project predicts house prices using Machine Learning techniques on the California Housing Dataset. The project covers the complete machine learning workflow from data preprocessing to model evaluation and prediction.

## Features

* Data Cleaning and Preprocessing
* Missing Value Handling using SimpleImputer
* Feature Scaling using StandardScaler
* Categorical Encoding using OneHotEncoder
* Pipeline and ColumnTransformer
* Cross Validation
* Hyperparameter Tuning using GridSearchCV
* House Price Prediction System

## Models Used

* Linear Regression
* Ridge Regression
* Lasso Regression
* Random Forest Regressor
* HistGradientBoostingRegressor

## Best Model

HistGradientBoostingRegressor

## Evaluation Metrics

* RMSE (Root Mean Squared Error)
* MAE (Mean Absolute Error)
* R² Score
